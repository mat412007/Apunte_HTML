Información y etiquetas de HTML.

  • HTML es un lenguaje de programación con el que se hacen las páginas web y aplicaciones.

  • HTML significa Hyper Text Markup Languaje.

    • Hyper Text significa texto dentro de texto, un link por ejemplo es un hypertext.

    • Markup Language significa que el texto es variado y dinámico, con imágenes, links, y muchas otras cosas.

  • Web Page es un documento escrito en HTML y traducido por un navegador web.

  • HTML está compuesto por etiquetas, y cada etiqueta tiene contenido.

  • <!DOCTYPE> define el tipo de documento y la versión del HTML usado.

  • <html> contiene cada elemento del HTML excepto !DOCTYPE, describe la página web.

  • <head> contiene info. que describe el documento, pero que no aparece con él.

  • <title> título de la página que aparece en el navegador, se inserta en head.

  • <body> su contenido es visible para el usuario, contenido principal de HTML.

  • <h1> primer nivel de heading de la página, son 6 niveles.

  • <p> su contenido es el de un párrafo de la página.

  • Tim Berners-Lee creó HTML, y lo lanzó en 1991.

  • La versión más nueva de HTML es HTML5, lanzada en 2008.

  • Características de HTML:

    • Un lenguaje simple y fácil.

    • Provee variedad de presentación y diseño.

    • Puede usarse en cualquier plataforma.

    • Permite insertar links, imágenes, sonidos y gráficos.

  • Al guardar un archivo HTML se le agrega la extensión .html.

  • Un documento HTML consiste en etiquetas y atributos.

    • Una etiqueta rodea el contenido que se le ponga y le da significado.

    • Un atributo provee información extra al elemento en donde se inserte.

  • Un elemento es el componente del archivo HTML, puede haber infinitos, es el conjunto de una etiqueta y su contenido.

  • Estructura de un elemento:

    • <p class="">This is the content<p/>

    • Contenido
    • Elemento
    • Atributo

  • Algunas etiquetas no necesitan etiquetas de cierre.

  • <br> significa break line, actua como la tecla Enter, inicia una nueva linea.

  • <hr> significa horizontal rule, traza una línea horizontalmente.

  • <style> y style="" puede usarse como elemento y atributo, se usa para insertar CSS en HTML directamente.

  • title="" muestra una descripción del contenido cuando el cursor está sobre él. CPU

  • Tanto comillas simples como dobles sirven para escribir atributos(' ' y " ")

  • Elementos sin etiqueta de cierre se llaman elementos vacios.

  • Un elemento puede contener otro elemento.

  • Por defecto, los elementos HTML se dividen en:

    • Blockl-level element: son bloques que abarcan todo el ancho de su alto, y pueden contener ambos tipos de elementos.

    • Inline element: abarcan la sección de su texto, nada más, no pueden contener bloques, pero sí líneas.

  • <div> crea secciones y agrupa contenidos, es un elemento en bloque.

  • <span> contiene frases en línea para darles estilos.

  • En HTML hay etiquetas que se usan para dar estilo al texto:

    • Etiquetas físicas: otorgan apariencia visual al texto.

    • Etiquetas lógicas: añaden valor lógico o semántico al texto.

  • Algunas etiquetas de formateo son:

    • <b> convierte el texto a negrita. Texto

    • <i> convierte el texto a cursiva. Texto

    • <u> subraya el texto. Texto

    • <sup> despliega texto arriba de la línea. 22

    • <sub> despliega texto debajo de la línea. H2O

    • <del> despliega texto tachado. Texto

  • Los HTML headings son títulos y subtítulos, de mayor a menor.
    • h1, h2, h3, h4, h5, h6.

  • Las etiquetas de frase que definen la estructura de un bloque de texto y sus semánticas son:
    • <em> hace lo mismo que <i>, pero con extra semánticas. Texto

    • <mark> destaca una parte importante del texto marcandoo con amarillo. Texto

    • <strong> remarca texto como <b>, pero le da importancia a la máquina también. Texto

    • <abbr> se usa para abreviar texto. HTML

    • <dfn> pone texto a definir en cursiva. Definición
