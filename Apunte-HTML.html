    Información general de HTML.

  • HTML es un lenguaje de programación con el que se hacen las páginas web y aplicaciones.

  • HTML significa Hyper Text Markup Language.

    • Hyper Text significa texto dentro de texto, un link por ejemplo es un hypertext.

    • Markup Language significa que el texto es variado y dinámico, con imágenes, links, y muchas otras cosas.

  • Web Page, una página web, es un documento escrito en HTML y traducido por un navegador web.

  • HTML está compuesto por etiquetas, y cada etiqueta tiene contenido.

  • La estructura inicial o básica de un documento HTML se compone de:

    • <!DOCTYPE> define el tipo de documento y la versión del HTML usado.

    • <html> contiene cada elemento del documento HTML, excepto <!DOCTYPE>, y describe la página web.

    • <head> contiene info. que describe el documento, pero que no aparece con él.

    • <title> título de la página que aparece en la pestaña del navegador, se inserta en head.

    • <body> su contenido es visible para el usuario, contenido principal de HTML.

    • <h1> primer nivel de heading de la página, son 6 niveles.

    • <p> su contenido es el de un párrafo de la página.

  • Tim Berners-Lee creó HTML, y lo lanzó en 1991.

  • La versión más nueva de HTML es HTML5, lanzada en 2008.

  • Características de HTML:

    • Un lenguaje simple y fácil.

    • Provee variedad de presentación y diseño.

    • Puede usarse en cualquier plataforma.

    • Permite insertar links, imágenes, sonidos y gráficos.

  • Al guardar un archivo HTML se le agrega la extensión .html.

  • Un documento HTML consiste en etiquetas y atributos.

    • Una etiqueta rodea el contenido que se le ponga y le da significado.

    • Un atributo provee información extra al elemento en donde se inserte.

  • Un elemento es el componente del archivo HTML, puede haber infinitos, es el conjunto de una etiqueta y su contenido.

  • Estructura de un elemento:

    • <p class="">This is the content<p/>

  • Algunas etiquetas no necesitan etiquetas de cierre.

  • Elementos sin etiqueta de cierre se llaman elementos vacios.

  • <br> significa break line, actua como la tecla Enter, inicia una nueva linea.

  • <hr> significa horizontal rule, traza una línea horizontalmente.

  • <style> y style="" puede usarse como elemento y atributo, se usa para insertar CSS en HTML directamente.

  • title="" muestra una descripción del contenido cuando el cursor está sobre él. CPU

  • Tanto comillas simples como dobles sirven para escribir atributos(' ' y " ")

  • Un elemento puede contener otro elemento.

  • Por defecto, los elementos HTML se dividen en:

    • Blockl-level element: son bloques que abarcan todo el ancho de su alto, y pueden contener ambos tipos de elementos.

    • Inline element: abarcan la sección de su texto, nada más, no pueden contener bloques, pero sí líneas.

  • <div> crea secciones y agrupa contenidos, es un elemento en bloque.

  • <span> contiene frases en línea para darles estilos.

  • En HTML hay etiquetas que se usan para dar estilo al texto:

    • Etiquetas físicas: otorgan apariencia visual al texto.

    • Etiquetas lógicas: añaden valor lógico o semántico al texto.

  • Algunas etiquetas de formateo son:

    • <b> convierte el texto a negrita. Texto

    • <i> convierte el texto a cursiva. Texto

    • <u> subraya el texto. Texto

    • <sup> despliega texto arriba de la línea. 22

    • <sub> despliega texto debajo de la línea. H2O

    • <del> despliega texto tachado. Texto

    • h1, h2, h3, h4, h5, h6.

    • <em> hace lo mismo que <i>, pero con extra semánticas. Texto

    • <mark> destaca una parte importante del texto marcandoo con amarillo. Texto

    • <strong> remarca texto como <b>, pero le da importancia a la máquina también. Texto

    Una imagen en HTML.

  • <img> despliega una imagen en la página web, no tiene contenido o etiqueta de cierre.

    • src="" contiene la fuente o camino de la imagen, nos dice donde buscarla.

    • alt="" contiene texto alternativo que aparece en el lugar de la imagen en caso de no ser desplegada.

    • width="" y height="" definen el ancho y el alto de la imagen, y ambos se usan en CSS.


    Tablas de HTML.

  • Una tabla en HTML se compone con muchas etiquetas y atributos:

    • <table> abre y cierra el cuadro de la imagen, en su interior están las etiquetas que definen la tabla.

    • <th> define el titulo(header) de la columna.

    • <td> define el contenido(data) de cada espacio de la tabla.

    • <caption> título de la parte de la tabla.

    • <colgroup> especifica un grupo de columnas para personalizarlas.

    • <col> se usa en <colgroup> para especificar la columna.

    • <thead> agrupa todos los <th>.

    • <tbody> agrupa el contenido general de la tabla, excepto la superior.

    • border="" especifica el borde de la tabla.

    • padding="" especifica el relleno de los <th> y <td>.

    • width="" especifica el ancho de la en % o px.

    • height=""especifica el alto del <table>, y solo puede ponerse en px.

    • border-collapse: collapse; colapsa los bordes en uno solo.

    • colspan="" se usa para que una columna tenga dos espacios para escribir.

    • rowspan="" se usa para que una fila tenga dos espacios para escribir.

    CSS en HTML.

  • Una vez que tenemos la estructura HTML, usamos CSS(Cascading Stylesheet).

  • CSS provee varios estilos y propiedades para aplicarlos, tienen un nombre y un valor, y cada uno se separa con (;).

  • Hay varios métodos para aplicar CSS:

    • Inline CSS: define propiedades CSS usando el atributo style="" en el elemento HTML que corresponda.

      • <h2 style='color: red; text-align: center;'></h2>

    • CSS Interno: define todas las propiedades de CSS juntas con <style> en la sección del <head>.

      • <head>br
      <style>
      text{
      color: red;
      }
      </style>
      </head>

    • CSS Externo: define todas las propiedades CSS en un archico .css aparte y después conectándolo con el HTML en el <head> usando <link>.

  • Algunas de las propiedades más usadas de CSS son:

    • background-color: define el color del ondo del elemento.

    • color: define el color del texto del elemento.

    • padding: define el tamaño del relleno de un elemento.

    • margin: define el espacio que hay entre el elemento y el resto de los componentes y el marco de la página.

    • border: define el ancho y el color del borde del elemento.

    • font-family: define el tipo de fuente de las letras de un elemento.

    • font-size: define el tamaño de letra de un elemento.

    • text-align: define la alineación horizontal del texto de un elemento.

  • HTML tiene un atributo llamado class="", se puede usar para dar función en JavaScript y CSS, podes usar la clase de HTML para darle estilos con <style> representándola con un punto (.), y podes poner la misma clase en más de un elemento para que su estilo sea el mismo.

  • Cada elemento puede tener más de una clase siempre y cuando estén separadas por un espacio, y diferentes elementos pueden tener la misma clase.

    • <style>
    .headings {
    color: red;
    }
    </style>


  • Hay otro atrbituo de HTML llamado id="", sus funciones son básicamente las mismas que las de class="", excepto que los id="" son únicos y no pueden repetirse.

  • A diferencia del class="", los id="" se representan con un #.

    • <head>
    <style>
    #text{
    color: red;
    }
    </style>
    </head>

    Página Web dentro de HTML.

  • <iframe> se usa para desplegar una página web anidada dentro del documento.

    • src="" contiene el URL de la página.

    • width="" y height="" especifican el alto y el ancho de la página insertada, en px o %.

    • frameborder: none; remueve el borde predeterminado de la ventana web.

    • target="" nos ayuda a conectar un <a> con la ventana del <iframe> mediante el uso de los atributos name="" para el <iframe> y el target="" para el <a>, y ambos deben ser iguales.

  • También puedes usar todos los códigos de arriba para vincular un video de YouTube en el <iframe>.

  • Algunos atributos de <iframe> son:

    • allowfullscreen="" permite que la ventana del <iframe> se expanda en toda la pantalla.

    • name="" define el nombre del <iframe>, y puede ser usado para vincularlo con JavaScript, <a>, <form>, <input>, <button>.

    JavaScript en HTML.

  • <script> se encarga de encerrar una sección en HTML en la que podés escribir funciones de script, principalmente JavaScript.

  • <script> puede ser ingresada en <head> o en <body>.

  • Para seleccionar un elemento de HTML, JavaScript usa document.getElementById() para seleccionar el elemento por su id="".

  • JavaScript actúa principalmente en los eventos, estos son básicamente acciones del usuario o de la página en sí, como un click del mouse o la carga de un página. HTML provee atributos para trabajar con JavaScript y realizar cierta acción.

  • Hay varios tipos de eventos:

    • Form events: reset, submit, etc.

    • Select events: text field, text area, etc.

    • Focus events: focus, blur, etc.

    • Mouse events: select, mouseup, mousedown, mousemove, click, dbclick, etc.

  • Algunos atributos de ventos son:

    • onClick="" cuando el usuario hace click en un elemento de <form> o un link.

    • onSubmit="" cuando el usuario entrega un <form> al servidor.

    • onLoad="" cuando una página se carga en el buscador.

    • onFocus="" cuando el usuario se enfoca en un campo <input>.

    • onSelect="" cuando el usuario selecciona el campo <input> de un <form>.

  • Podes cambiar el contenido de un elemento HTML de la siguiente manera:

    • document.getElementById("id").innerHTML = "Hello World!";.

  • Podes cambiar el estilo de un elemento HTML de la siguiente manera:

    • document.getElementById("id").style.color = "red";

  • Podes cambiar el atributo de un elemento HTML de la siguiente manera:

    • document.getElementById("id").src = "pic.png"

  • Al tener muchos archivos HTML que necesiten tener un <script>, podes poner todos los <script> en un solo archivo de JavaScript, y luego vincularlo con los HTML de la siguiente manera.

    • <script> type="text/javascript" scr="./script.js"</script>.

  • <noscript> se usa para escribir <script> deshabilitado, su contenido no aparece en el navegador.

    • Direcciones de carpeta de HTML.

  • Una ruta de archivo HTML describe la localización de un archivo en una carpeta web.

  • Las rutas de archivo son como una dirección de un archivo para un navegador web.

  • Podemos usar estas rutas para vincular con HTML algunas fuentes externas para darle más dinámica, como imagenes, archivos CSS, archivos JavaScript, etc.

  • Los diferentes tipos de maneras de especificar rutas de archivos son con los atributos src="" y href="", de la siguiente manera:

    • src="picture.jpg" especifica que la imagen está en la misma carpeta de la misma página.

    • src="images/picture.jpg" especifica que la imagen está ubicada en la carpeta images dentro de tu carpeta actual.

    • src="/images/picture.jpg" especifica que la imagen está en la carpeta images en las raíces de la página actual.

    • src="../picture.jpg" especifica que la imagen está ubicada en la carpeta un nivel más arriba que la actual.

  • Estas rutas se pueden usar para vincular:

    • Página Web.

    • Imágenes.

    • Hojas de estilo.

    • JavaScript.

  • Hay dos tipos de rutas:
    • Rutas de archivo absolutas:
    Rutas absolutas especifican la dirección URL completa.

    Rutas de archivo relativas:
    Rutas relativas especifican una carpeta relacionada con la página actual.

    Head de HTML.

  • El elemento <head> contiene metadata, data que modifica otra data, como links, títulos, estilos, y otra información meta.

  • Se inserta entre <html> y <body>.

  • Su contenido no se muestra en el navegador. Su contenido solo modifica la data que si se muestra.

  • Algunas etiquetas usadas en <head>:

    • <title>:
      Este elemento se usa para definir el título del documento. Su titulo se muestra en la barra de búsqueda del navegador al acceder a la página. Este título se usa para buscar la página desde cualquier navegador desde cualquier dispositivo.

    • <style>:
      Este elemeto se usa para darle estilo y apariencia a la página de HTML. Puede contener propiedades CSS para esa página únicamente, para aplicarla a muchas páginas a la vez, se insertan en un archivo de '.css' aparte.

    • <meta>:
      Este elemento se usa para especificar la descripción de la página, palabras clave, autores, y otra metadata de la página. Metadata es usada por navegadores para nivelar mejor las páginas web.
      charset="" especifica la codificación de los carácteres, usualmente se usa UTF-8 como su valor, indicando que puede imprimir cualquier lenguaje.
      name="description" se usa para mejorar la búsqueda de la página, mencionando una descrioción de su contenido, para eso usamos también content="".
      name="keywords" es un valor que se usa para proveer palabras claves para un buscador, se usa también con content="".
      name="author" este valor especifica el nombre de la persona que escribió el contenido de la página, es útil para obtener información del autor rápidamente, tambien se usa content="" en él.
      http-equiv="refresh" se usa para establecer un lapso en el que la página se reinicia cuando se cumple, el temporizador se establece en segundos, con el atributo content="", además, si añadis un URL delante del temporizador con un ";", se irá a esa página en su lugar.
      En HTML5, <meta> también se puede usar para controlar el Viewport, se refiere a la parte del documento que usted está viendo, la cual es actualmente visible en su ventana, que según el dispositivo que uses puede ir cambiando dependiendo de su tamaño. Para setearlo usamos el atributo name="viewport", luego usamos content="width=device-width, initial-scale=1.0", width=device-width se usa para colocar el ancho de la página al ancho del dispositivo. initial-scale=1.0 se usa para colocar el zoom inicial de la página al ser abierta.

    • <link>:
      Este elemento se usa para vincular una hoja de estilos a tu página web. Contiene 2 atributos:
      href="" contiene la ruta del archivo, su ubicación.
      rel="" este indica que el archivo es un stylesheet.

    • <script>:
      Este elemento se usa para aplicar código JavaScript en esa misma página o para vincular un archvio JavaScript aparte con la página usando src="".

    • <base>:
      Este elemento se usa para especificar el URL base y objetivo para todos los URL del resto de la página, con el atributo href="".

    Codificación URL de HTML.

  • URL significa Uniform Resource Locator, básicmente es una dirección web.

  • Contiene una dirección IP, Internet Protocol, como i.e.195.201.68.81. Sin embargo, es más fácil recordar una dirección URL con palabras más que con números.

  • Sintaxis de un URL:

    • scheme://prefix.domain:port/path/filename

    • scheme se usa para definir el tipo de servicio de Internet, usualmente http o https.

    • prefix se usa para definir el dominio del dominio que queremos seleccionar, el prefijo, o sea "www".

    • domain se usa para definir el nombre del dominio de Internet, como javaTpoint.com.

    • port se usa para definir el número de puerto en el anfitrión, por defecto es 80.

    • path se usa para definir el camino en el servidor.

    • filename se usa para definir el nombre del documento o de una fuente.

  • Una lista de los esquemas(schemes) más comunes usados en URL son:

    • http: (HyperText Transfer Protocol), páginas web comúnes, no protegidas.

    • https: (Secure HyperText Transfer Protocol), páginas web seguras, protegidas.

    • ftp: (File Transfer Protocol), subiendo o bajando archivos.

  • Codificación URL se usa para convertir carácteres que no son de ASCII a un formato que puede ser usado en el Internet, esto se debe a que un URL es enviado por Internet usando únicamente el set de carácteres ASCII, si contiene carácteres que no son ASCII, deben ser convertidos.

  • En la codificación de URL, los carácteres que no son de ASCII son reemplazados por "%" y un número hexadecimal adelante.

  • La codificación URL reemplaza espacios con "+" o con "%20".

    Drag and Drop

    HTML Drop and Drag (DnD) es propio de HTML5. Es una poderosa interfaz del usuario que se puede usar para copiar, reordenar y eliminar, todo con la ayuda del mouse. Funciona al mantener presionado el botón del mouse sobre un elemento y arrastrarlo hacia otra posición al soltar el botón.
    Los eventos de DnD son:

    • ondrag="" se ejecuta cuando el mouse se mueve mientras el objeto está siendo arrastrado.

    • ondragstart="" se activa cuando el usuario comienza a arrastrar el objeto.

    • ondragenter="" se activa cuando el usuario mueve el elemento que está siendo arrastrado sobre una sección de caída válida.

    • ondragover="" se activa cuando el mouse se mueve sobre un elemento válido para el arrastre.

    • ondragleave="" este evento se activa cuando el mouse deja un elemento.

    • ondrop="" se activa al final, cuando termina la operación de arrastre.

    • ondragend="" se activa cuando el usuario libera el elemento tras haber sido arrastrado y tras completar la operación de arrastre.

    • draggable="true" establece que el elemento es arrastrable.
